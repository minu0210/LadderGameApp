# LadderGame
---
### 개요
 - 결과물이 있는 개발을 하기 위하여 주제를 찾던 중, 사다리타기라는 과제를 받아서 기능이라도 구현해보자는 생각으로 진행하게 되었다.
### 개발 기간
 - 2023-11-27 ~ 2023-12-15
### 스킬
 - WPF
 - C#
### 리뷰
 - 처음 과제를 받았을 땐 C#의 기본 문법 정도만 알고 WPF는 들어본 적도 없어서 막막했지만, 하다보니 의외로 무난하게 개발하여서 괜찮았다.
### 부족한 부분 및 프로그램 개선점
 - 객체지향적인 기획이 어려워서 초기 구조가 여러번 수정되었다.
 - 디자인적인 요소를 구상하기 어려웠다.
 - 게임에 대한 소개 및 규칙과 같은 내용이 부족하다.
 - UI가 불친절하다.
 - 사다리 경로를 그리는 도중 다른 사다리의 경로가 그려지게 변경되지 않는다.
